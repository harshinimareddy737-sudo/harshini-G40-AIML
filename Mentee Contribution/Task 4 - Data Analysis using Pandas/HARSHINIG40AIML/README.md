# Customer Churn EDA Project



## Files in this folder
- `Customer_Churn_Dataset_[YourName].csv` — original raw dataset
- `Cleaned_Customer_Churn_Dataset_[YourName].csv` — cleaned dataset after EDA
- `Customer_Churn_EDA_[YourName].ipynb` — full EDA notebook (data cleaning, 
  univariate/bivariate analysis, 8+ visualizations, correlation analysis)
- `Customer_Churn_EDA_[harshini].pdf` — written report with findings 
  and business recommendations

## Dataset
Bank customer churn dataset (10,000 rows, 14 columns), sourced from a public 
GitHub mirror of the Kaggle "Churn Modelling" dataset. Target column: `Exited`.

## Summary
20.37% of customers churned. Key churn drivers identified: number of 
products held, geography (Germany highest), customer age, account activity 
status, and gender. Full details in the EDA report.
